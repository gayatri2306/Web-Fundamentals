JavaScript Control Flow Visualizer

1. The For Loop

Best for: Running code a specific number of times.

for (let i = 1; i <= 3; i++) { ... }

2. The While Loop

Best for: Running until a condition is no longer met. (Checks condition before running)

while (count < 3) { ... }

3. The Do...While Loop

Best for: Ensuring the code runs at least once.

do { ... } while (condition);
